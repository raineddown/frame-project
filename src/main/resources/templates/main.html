公司员工操作平台

1. 简介

这是一套基于spring boot、shiro、jwt、redis、swagger2、mybatis、layui 后台管理系统，包含权限控制的方式为 RBAC(shiro实现)、签发JWT（无状态token）过期自动刷新，redis校验判断token是否有效，数据全程 ajax 获取，封装 ajax 工具类、菜单无线层级的展示和实现。

2. 系统功能

  1. 用户管理：用户是系统操作者，该功能主要完成系统用户配置。
  2. 部门管理：配置系统组织机构（公司、部门、小组），layui组件实现树结构展现。
  3. 菜单管理：配置系统菜单，操作权限，按钮权限标识等。
  4. token管理：对用户的操作签发JWT类型token，用于前端获取用户信息以及后端验证登录。
  5. 角色管理：shiro角色菜单权限分配、设置角色按机构进行数据范围权限划分。
  6. 接口管理：swagger根据业务代码自动生成相关的 api 接口文档。
  7. SQL 监控：druid对系统使用的 sql 进行监控，可快速查询运行效率。
  8. 日志管理：对用户的操作进行记录。
